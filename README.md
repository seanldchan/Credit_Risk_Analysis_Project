# Credit Risk Analysis Project

This project builds a maching learning to predict the probability of loan default (PD), assisting lenders to identify high-risk applicants and reduce expected credit losses.

## Dataset
- Source: Kaggle / public lending dataset
- Rows: 32,000
- Target: loan_status (default / non-default)

## Tools Used
- Python
- pandas
- numpy
- scikit-learn
- matplotlib / seaborn
- Logistic Regression

## Methodology
1. Data cleaning
2. Missing value treatment
3. Feature engineering
4. Train-test split
5. Model training
6. Cross-validation
7. Performance evaluation

## Business Impact
The model can help reduce bad debt exposure by flagging high-risk applicants before approval.
